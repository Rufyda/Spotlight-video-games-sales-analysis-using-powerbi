## Description
This project analyzes **video game sales from 1980 to 2020**, focusing on **yearly sales trends** and **sales by genre**. The data was cleaned and prepared using **SQL** and visualized in an interactive **Power BI dashboard**.

## Features
- Yearly sales analysis
- Sales breakdown by genre
- Interactive Power BI dashboard with tooltips and filters


## Technologies Used
- Figma UX
- Power BI
- Data Visualization
- Data Analysis

## How to Use
1. Open FILE in Power BI.
2. Explore the interactive dashboard to view sales trends and genre breakdowns.
3. Use filters and tooltips for deeper insights.
